Base environment for run TypeScript of use ts-node.

## Creation Process of this Repository.

```
  # create package.json
$ yarn init

  # install TypeScript
$ yarn add typescript

  # install ts-node
$ yarn add ts-node

  # create tsconfig.json
$ npx tsc --init

  # create src/main.ts
$ mkdir src && touch src/main.ts
```

## Run Code of main.ts

```
$ yarn main
```
